# Backup base de donnee

```bash

SELECT *
INTO OUTFILE 'C:\backup'
FIELDS TERMINATED BY ','
ENCLOSED BY '"'
LINES TERMINATED BY '\n'
FROM information_schema.tables
WHERE table_schema = 'prototype';

```
